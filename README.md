# AA-Editor 

色付きのアスキーアートを作成するためのエディターです。

## AEファイル　データ形式 

00480000000100000000

|文字コード(UTF16)|背景色|文字色|Y座標|X座標|
|:-:|:-:|:-:|:-:|:-:|
|0048|0000|0001|0000|0000|
|H|BLACK|BLUE|0|0|
